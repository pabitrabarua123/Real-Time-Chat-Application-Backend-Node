# 💬 Real-Time Chat Application

A full-stack real-time one-to-one chat application built using React, Node.js, Express, and Socket.IO.

---

## 🚀 Features

- 🔐 User authentication (Login/Register)
- 💬 Real-time messaging with Socket.IO
- 👥 One-to-one chat system
- 🟢 Online/offline user status
- 📱 Responsive UI
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Axios
- Socket.IO Client

### Backend
- Node.js
- Express.js
- Socket.IO

### Database
- MongoDB / MySQL (based on your setup)
